# Asset 3D

Inserire qui i modelli 3D in formato `.obj`, `.fbx` o `.glb`.

Il nome del file deve corrispondere al nome inglese dell'oggetto
restituito dal modello LLM.

## Esempi

| File           | Oggetto        |
|----------------|----------------|
| `table.obj`    | Tavolo         |
| `chair.obj`    | Sedia          |
| `lamp.obj`     | Lampada        |
| `desk.obj`     | Scrivania      |
| `sofa.obj`     | Divano         |
| `bookshelf.obj`| Libreria       |

## Fonti gratuite

- [Sketchfab](https://sketchfab.com/features/free-3d-models)
- [BlendSwap](https://www.blendswap.com)
- [Poly Haven](https://polyhaven.com)
- [OpenGameArt](https://opengameart.org)

## Note

I file `.obj` grandi non sono inclusi nel repository.
Aggiungere questa cartella a `.gitignore` se i modelli sono pesanti,
oppure usare Git LFS per tracciare file binari di grandi dimensioni.
